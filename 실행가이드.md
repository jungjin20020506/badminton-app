# 📖 실행가이드 — 처음 쓰는 사람용

컴퓨터를 잘 몰라도 이 순서대로만 하면 됩니다. **1단계는 컴퓨터당 딱 한 번만** 하면 되고,
그 다음부터는 **"실행하기.bat" 더블클릭**만 하면 됩니다.

---

## 1단계. Python 설치 (컴퓨터당 최초 1회)

이 프로그램은 Python(파이썬)이라는 무료 프로그램 위에서 동작합니다.

1. 인터넷 브라우저에서 **https://www.python.org/downloads/** 접속
2. 노란색 **"Download Python 3.x.x"** 버튼 클릭 → 설치 파일 다운로드
3. 다운로드한 파일을 더블클릭해 설치를 시작
4. ⚠️ **가장 중요!** 설치 첫 화면 맨 아래에 있는
   **"Add python.exe to PATH"** 체크박스를 **반드시 체크** ✅
5. **"Install Now"** 클릭 → 설치 완료까지 기다리기 → "Close"

> 이미 Python이 설치되어 있다면 이 단계는 건너뛰세요.
> (확인 방법: "실행하기.bat"를 더블클릭했을 때 프로그램이 켜지면 설치되어 있는 것입니다.)

## 2단계. 엑셀 기능 설치 (최초 1회, 인터넷 필요)

체크시트·주간보고서·히스토리를 **엑셀 파일로 내려받는 기능**에 필요합니다.

1. 이 폴더에 있는 **`최초설치.bat`** 를 더블클릭
2. 검은 창에 글자가 올라가다가 "설치가 끝났습니다!" 가 나오면 완료 → 아무 키나 눌러 닫기

> 이 단계를 건너뛰어도 검증 기능 자체는 모두 동작합니다.
> 엑셀 다운로드 버튼을 누를 때만 설치 안내가 표시됩니다.

## 3단계. 프로그램 실행 (매번 이렇게 시작)

1. 이 폴더에 있는 **`실행하기.bat`** 를 더블클릭
2. 검은 창이 뜨고 1~2초 뒤 **브라우저가 자동으로 열립니다** → 바로 사용하면 됩니다
3. ⚠️ **검은 창을 닫으면 프로그램이 꺼집니다.** 사용하는 동안은 최소화만 해 두세요.

끝낼 때는 검은 창을 닫으면 됩니다. (데이터는 자동 저장되어 있으니 안심하세요.)

## 자주 겪는 문제 (이것만 알면 됩니다)

**Q. 버튼을 눌렀더니 "서버에 연결할 수 없습니다"라고 떠요.**
→ 프로그램(검은 창)이 꺼져 있는 상태입니다. `실행하기.bat`를 다시 더블클릭한 뒤,
   **자동으로 새로 열리는 화면**에서 사용하세요. (예전에 열려 있던 탭은 닫아도 됩니다.)

**Q. "포트가 사용 중"이라는 메시지가 보여요.**
→ 걱정하지 않아도 됩니다. 프로그램이 **자동으로 다음 번호의 포트로 바꿔서** 실행됩니다.

**Q. 엑셀 다운로드가 안 되고 openpyxl 설치 안내가 떠요.**
→ 2단계(`최초설치.bat`)를 아직 안 한 것입니다. 인터넷이 되는 상태에서 한 번 실행해 주세요.

**Q. 검은 창을 실수로 닫았어요. 데이터가 날아갔나요?**
→ 아닙니다. 모든 데이터는 `data\quality.db` 파일에 자동 저장됩니다. 다시 실행하면 그대로 있습니다.

**Q. 데이터를 안전하게 보관하고 싶어요.**
→ 홈 화면 맨 아래 **"💾 백업 파일 다운로드"** 버튼을 주기적으로 눌러 받은 파일을
   USB나 사내 서버에 보관하세요. 복원할 때는 받은 파일 이름을 `quality.db`로 바꿔
   `data` 폴더에 덮어쓰면 됩니다.

**Q. 이 폴더가 OneDrive 안에 있는데 괜찮나요?**
→ 대체로 괜찮지만, 드물게 OneDrive 동기화가 DB 파일을 잠가 오류가 날 수 있습니다.
   문제가 반복되면 폴더를 `C:\KNK` 같은 OneDrive 밖 경로로 옮기는 것을 권장합니다.

## 명령어로 직접 실행하고 싶다면 (선택)

```bash
python run.py                 # 기본 실행 (http://127.0.0.1:8000, 브라우저 자동 열림)
python run.py --port 9000     # 포트 직접 지정
python run.py --no-browser    # 브라우저 자동 열기 끄기
```
